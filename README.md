# Axis select
blender addon

# Description
This blender addon can select vertices according to axis.

- orientation
  - `Global`
  - `Local`
- axis
  - `-X`
  - `+X`
  - `-Y`
  - `+Y`
  - `-Z`
  - `+Z`
- offset
  - from axis

# Usage

1. Activate this addon.
1. Switch `Edit Mode`.
1. `Select > Select according to axis.`
1. You can update orientation, axis and offset.
